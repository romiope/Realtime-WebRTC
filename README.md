# Voice

Real-time voice chat using OpenAI's Realtime API and WebRTC.

## Setup

```bash
bun install
```

Create `.env.local`:

```
OPENAI_API_KEY=sk-...
```

## Run

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000).
